** Quotation marks("") and square brackets([]) are not to be included when running the commands.

- Create a new React App with the command [npm init vite@latest any_name_you_want]

- Replace these files after creating a new react app [src, package.json, index.html]
  with the downloaded files[src, package.json, index.html]

- Open any terminal of your choice, most likely the IDE terminal, make sure the path is set to the 
  created react app

- Run "npm install" [ie: after replacing the files]. This will install all the dependencies needed for
  the project

- Run "npm run dev", this will start the project and generate a link to the project on the browser.
